API ROUTES:

(REGISTERD MANAGEMENT):

(AUTH MODULE):

(PROFILE MODULE):
Get Profile: GET: https://pet-adoption-server-sooty.vercel.app/api/profile
Update Profile: PUT: https://pet-adoption-server-sooty.vercel.app/api/profile
